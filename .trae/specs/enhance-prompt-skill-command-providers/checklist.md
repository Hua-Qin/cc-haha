# Checklist

## 提示词优化功能
- [ ] 输入框工具栏中存在"优化提示词"按钮，点击后触发 AI 优化
- [ ] 优化时读取当前会话上下文（消息历史）进行上下文感知优化
- [ ] 优化完成后输入框文本被替换为优化后文本
- [ ] 设置 → 通用 中存在"AI 提示词优化"配置区块（开关、优化指令、模型、temperature）
- [ ] 自定义优化配置修改后，后续优化操作使用新配置
- [ ] 优化操作被记录到 Trace 系统（包含原始/优化后提示词、时间、上下文）
- [ ] 优化产生的 tokens 消耗被准确统计到会话成本（`cost-tracker.ts` 的 `addToTotalSessionCost`）

## AI 默认提示配置
- [ ] 所有默认提示配置（系统提示词、角色提示词、权限提示词）经过审计，无逻辑矛盾
- [ ] 提示词切换功能（含"跳过模式" / overrideSystemPrompt）经验证稳定可用
- [ ] AI 思考阶段实时流式展示思考内容（ThinkingBlock），而非仅显示"思考中"占位
- [ ] 所有触发思考的代码路径均正确发出 `{ type: 'thinking'; text }` 流式事件

## 服务商配置
- [ ] 所有服务商（包括系统默认/官方服务商）均支持编辑操作
- [ ] 所有服务商（包括系统默认/官方服务商）均支持删除操作
- [ ] 删除服务商前显示确认对话框，包含服务商名称和影响说明
- [ ] 冗余/过期服务商预设已从 `providerPresets.json` 中清理

## 技能管理功能
- [ ] 设置 → 技能管理 中存在"在线安装技能"入口
- [ ] 安装流程包含：技能源验证、权限检查（allowed-tools 审查）、完整性校验（SKILL.md 解析）
- [ ] 安装过程提供清晰的进度反馈
- [ ] 安装成功后技能列表自动刷新
- [ ] 安装失败时显示明确原因，不写入文件到技能目录
- [ ] 提供成功/失败提示及后续操作指引

## 命令管理系统
- [ ] 输入框中斜杠("/")触发命令列表稳定可用
- [ ] 设置中存在"命令管理"配置模块（Settings Tab）
- [ ] 命令管理支持置顶设置，置顶状态持久化
- [ ] 置顶命令在输入框命令建议列表中优先显示（排在最前）
- [ ] 命令按分类（内置/技能/插件/MCP）在输入框中准确标注或分组显示
- [ ] 命令分类在设置管理界面中准确显示
